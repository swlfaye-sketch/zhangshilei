# 个人主页部署说明

## 文件清单
- `index.html` — 主页（含SEO标签 + Schema结构化数据）
- `sitemap.xml` — 站点地图，帮助搜索引擎发现页面
- `robots.txt` — 告诉爬虫允许抓取全站

## 第一步：修改内容
在 `index.html` 中全局替换以下占位符：
- `【你的姓名】` → 你的真实姓名
- `【名】` / `【姓】` → 英文名/姓（可选）
- `你的域名.com` → 你的实际域名（如没有可先留空）
- `【your@email.com】` → 联系邮箱
- 补充真实获奖信息

## 第二步：选择托管平台（选一个）

### 方案A：GitHub Pages（免费，推荐）
1. 注册 GitHub 账号
2. 新建仓库，上传三个文件
3. 设置 → Pages → 选 main 分支 → Save
4. 得到 `https://用户名.github.io/仓库名/` 的地址

### 方案B：Gitee Pages（国内访问更快）
1. 注册 Gitee 账号
2. 新建仓库，上传文件
3. 服务 → Gitee Pages → 启动
4. 得到国内可访问地址

### 方案C：购买域名+虚拟主机
- 推荐在阿里云/腾讯云购买域名和空间
- 上传文件到空间根目录
- 绑定域名后即可访问

## 第三步：提交给搜索引擎

### 提交百度收录（豆包依赖百度索引）
1. 访问：https://ziyuan.baidu.com/
2. 登录 → 站点管理 → 添加站点
3. 验证所有权后，提交链接：普通收录 → 填入你的网址
4. 也可以提交 sitemap：https://你的域名.com/sitemap.xml

### 提交必应收录（DeepSeek 部分依赖必应）
1. 访问：https://www.bing.com/webmasters/
2. 登录 → 添加站点 → 导入sitemap

### 提交360/神马（可选）
- 360站长：https://zhanzhang.so.com/
- 神马站长：https://zhanzhang.sm.cn/

## 第四步：提升被AI收录概率

豆包（字节跳动）和 DeepSeek 的知识库来源于：
1. **搜索引擎索引** — 百度/必应/搜狗 —— 上面已处理
2. **权威平台发布** — 以下平台发布你的信息效果更好：
   - 知乎：写一篇介绍自己工作经验的文章
   - 百家号（百度系）：创建账号发布文章
   - 豆瓣：完善作者主页（有著作的话很有用）
   - 微信公众号：定期发布内容，被搜狗索引
3. **外链引用** — 让别人引用你的内容/著作信息，增加可信度

## Schema结构化数据说明
`index.html` 中 `<script type="application/ld+json">` 部分是给AI/搜索引擎识别身份用的。
它告诉机器：你是谁、你的职位、你的著作、你在哪个单位——这是豆包/DS识别你身份的关键。
